1. Перейдите в папку `e-bee`. `cd e-bee`
2. Запустите docker 
3. Выполните команду `browser-sync start --config bs-config.js`
4. Откройте  Local: http://localhost:3000 или External: http://192.168.3.12:3000 адрес
